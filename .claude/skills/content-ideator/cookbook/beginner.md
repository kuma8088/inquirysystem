# 初心者向けコンテンツ戦略

入門者・キャリアチェンジ希望者をターゲットにしたコンテンツの追加観点。

## 初心者向けの特徴

| 要素 | 重視すべき点 |
|:---|:---|
| **敷居の低さ** | 「自分にもできそう」と思わせる |
| **ステップバイステップ** | 迷わない手順 |
| **つまずきポイント** | 初心者がハマる箇所を先回り |
| **励まし** | 挫折しないサポート |

## 推奨プラットフォーム

| Platform | 優先度 | 理由 |
|:---|:---:|:---|
| **Zenn** | ⭐⭐⭐⭐⭐ | 入門記事が人気 |
| **Qiita** | ⭐⭐⭐⭐ | 検索流入が多い |
| **note** | ⭐⭐⭐⭐ | キャリア系と相性◎ |
| **YouTube** | ⭐⭐⭐⭐ | 動画で手順を見せる |
| **Twitter/X** | ⭐⭐⭐ | 学習仲間とつながる |

- [前提知識]

## この記事で得られること
- ✅ [学べること1]
- ✅ [学べること2]

[中学生でもわかる説明]

## なぜ [技術] を学ぶべきか
[学ぶメリット、キャリアへの影響]

## 学習ロードマップ
```

## 初心者の気持ちに寄り添う表現

### 避けるべき表現

```
❌ 「簡単です」「すぐできます」
❌ 「当然ですが」「知っていると思いますが」
❌ 「普通は〜」「常識的に〜」
```

### 使うべき表現

```
✅ 「最初は難しく感じるかもしれませんが」
✅ 「私も最初はここでつまずきました」
✅ 「ゆっくり進めていきましょう」
✅ 「わからなくても大丈夫です」
```
