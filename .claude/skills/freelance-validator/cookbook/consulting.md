# コンサルティング事業の追加評価軸

コンサル・アドバイザリー系アイデアを評価する際の追加観点。

## 追加調査項目

Web検索で以下を追加調査：
- 「[分野] コンサルタント 料金」
- 「[分野] 顧問 相場」
- 「[スキル] フリーランス 単価」

## コンサル固有の評価軸

| 軸 | 観点 | 重要度 |
|:---|:---|:---:|
| **専門性の証明** | 実績・資格で信頼を得られるか | ⚠️ 高 |
| **単価設定** | 時給換算でいくら取れるか | ⚠️ 高 |
| **リピート性** | 継続契約になりやすいか | ⚠️ 高 |
| **営業負荷** | 顧客獲得の難易度 | 中 |
| **スケール限界** | 1人で対応できる上限 | 中 |

## 単価の目安

| レベル | 時給目安 | 月収（稼働80h） |
|:---|:---|:---|
| 初級（経験1-2年） | ¥3,000-5,000 | ¥24-40万 |
| 中級（経験3-5年） | ¥8,000-15,000 | ¥64-120万 |
| 上級（経験10年+） | ¥20,000-50,000 | ¥160-400万 |
| エキスパート | ¥100,000+ | ¥800万+ |

## 危険信号 🚨

以下に該当する場合は **Feasibility を1段階下げる**：

- [ ] 実績がゼロの分野
- [ ] 資格が必須（弁護士、税理士等）
- [ ] 大手コンサルと直接競合
- [ ] 顧客単価が低い（¥5,000/h 以下）
- [ ] 市場が縮小傾向

## 成功パターン

1. **ニッチ専門家**: 「○○専門」で第一想起を取る
2. **実績ショーケース**: ブログ・登壇で専門性を証明
3. **パッケージ化**: 時間売りから成果物売りへ
4. **コミュニティ形成**: 顧客同士をつなげて価値を増幅

## スケール戦略

労働集約型から脱却する方法：
- 教材・テンプレート販売
- グループコンサル
- オンラインコース
- 認定制度・フランチャイズ
